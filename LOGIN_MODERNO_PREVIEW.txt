╔═════════════════════════════════════════════════════════════════════════════╗
║                         FLUXO DE INTERAÇÃO                                 ║
╠═════════════════════════════════════════════════════════════════════════════╣
║                                                                             ║
║  1. PÁGINA CARREGA                                                          ║
║     └─ Logo entra com scale 0.9 → 1 (500ms)                               ║
║     └─ Form entra com opacity 0 → 1 (600ms, delay 300ms)                  ║
║     └─ Imagem de fundo em scale 1.1 → 1 (800ms)                           ║
║                                                                             ║
║  2. USUÁRIO DIGITA EMAIL                                                   ║
║     └─ Input border: white/10 (normal)                                     ║
║     └─ Input border: yellow-400/50 (focus)                                ║
║     └─ Ícone: gray (normal) → yellow (focus)                              ║
║     └─ Input box-shadow: subtle yellow glow                                ║
║                                                                             ║
║  3. USUÁRIO DIGITA SENHA                                                   ║
║     └─ Mesmo efeito do email                                               ║
║     └─ Ícone eye toggle no lado direito                                    ║
║     └─ Click eye: show/hide password                                       ║
║                                                                             ║
║  4. USUÁRIO CLICA BOTÃO "ENTRAR"                                           ║
║     └─ Hover: scale 1 → 1.02, shadow intensifica                           ║
║     └─ Tap: scale 1.02 → 0.98 (click feedback)                            ║
║     └─ Spinner aparece: "Processando..."                                   ║
║     └─ Botão desabilitado (opacity-50)                                     ║
║     └─ Aguarda resposta Supabase...                                        ║
║                                                                             ║
║  5. SUCESSO                                                                 ║
║     └─ Mensagem verde: "✅ Acesso concedido"                              ║
║     └─ App navega para dashboard                                           ║
║                                                                             ║
║  6. ERRO                                                                    ║
║     └─ Mensagem vermelha: "Email ou senha inválidos"                      ║
║     └─ Spinner desaparece                                                  ║
║     └─ Usuário pode tentar novamente                                       ║
║                                                                             ║
║  7. TOGGLE PARA SIGNUP                                                     ║
║     └─ Click "NOVO? CRIAR CONTA"                                          ║
║     └─ isSigningUp muda para true                                          ║
║     └─ Texto muda para "Criar Conta" / "Junte-se..."                      ║
║     └─ Botão muda para "Criar Conta"                                       ║
║     └─ Toggle volta para "Já tem conta? Entrar"                            ║
║                                                                             ║
║  8. SIGNUP SUCESSO                                                          ║
║     └─ Mensagem verde: "✅ Conta criada! Verifique seu email."            ║
║     └─ Auto-toggle para login após 3s                                      ║
║                                                                             ║
║  9. SIGNUP ERRO                                                             ║
║     └─ Mensagem vermelha com motivo                                        ║
║     └─ Usuário pode tentar novamente                                       ║
