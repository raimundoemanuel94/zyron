---
description: Como preparar o projeto para análise no NotebookLM
---

1. Execute a ferramenta de análise de projeto para coletar o estado atual.
2. Gere o arquivo `notebooklm_context.md` contendo:
   - Esquema do Supabase.
   - Lógica do `useSyncWorkout`.
   - Configuração do `vite.config.js`.
3. Verifique se não há tokens expostos.
4. Entregue o markdown para o usuário.
